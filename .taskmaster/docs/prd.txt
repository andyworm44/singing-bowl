# 靜心缽 (Singing Bowl) 冥想應用程序

## 產品願景
創建一個簡單而優雅的冥想應用程序，通過傳統的靜心缽聲音來幫助用戶達到平靜的心境。

## 核心功能

### 1. 靜心缽互動
- 點擊時顯示 1.1x 縮放動畫效果
- 播放真實的靜心缽聲音
- 顯示優雅的漂浮文字動畫

### 2. 自動敲擊功能
- 每3秒自動敲擊一次靜心缽
- 可在設置中開啟/關閉
- 保持聲音的自然感和連續性

### 3. 統計追踪
- 記錄每日冥想時間
- 顯示連續冥想天數
- 計算每週冥想總時間
- 使用 AsyncStorage 保存數據

### 4. 用戶界面

#### 頂部導航欄
- 左側：設置按鈕
- 中間：每日靜心文字 (24px)
- 右側：統計按鈕

#### 模態框設計
- 設置模態框：包含自動敲擊開關
- 統計模態框：顯示冥想數據
- 每日靜心模態框：顯示今日冥想時間、連續天數和每週總時間
- 所有模態框使用半透明黑色背景
- 統一的關閉按鈕樣式

### 5. 視覺設計
- 使用金色 (#FFD700) 作為主題色
- 確保所有按鈕和文字有統一的樣式
- 優雅的動畫過渡效果
- 清晰的視覺層次

## 技術要求
- 使用 React Native 和 Expo
- 實現響應式設計
- 確保聲音播放的穩定性
- 優化性能和動畫流暢度

## 用戶體驗目標
- 簡單直觀的操作
- 平靜舒適的視覺體驗
- 流暢的動畫過渡
- 可靠的數據保存
